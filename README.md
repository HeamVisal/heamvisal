## Featured Projects

### Harbor Property Rent Analysis
> *Machine Learning · Web Scraping · Predictive Modeling*

An end-to-end data pipeline that scrapes rental property listings, performs systematic data cleaning and exploratory analysis, and applies machine learning models to estimate rental prices.

**Key highlights:**
- Automated data collection with Selenium and BeautifulSoup
- Feature engineering and EDA on real estate data
- Regression models including XGBoost for rent price prediction

**Stack:** `Python` `Selenium` `BeautifulSoup` `Pandas` `Scikit-learn` `XGBoost`

---

### CardScan using ESP8266
> *Embedded Systems · IoT · RFID · Cloud Integration*

An RFID-based attendance and access control system built around the ESP8266 microcontroller. Card scans are logged in real time to Google Sheets via Wi-Fi using REST API calls.

**Key highlights:**
- RC522 RFID reader integration with ESP8266
- Embedded web server for device configuration
- Real-time logging to Google Sheets via Google Apps Script REST API

**Stack:** `Arduino` `ESP8266` `RFID (RC522)` `REST API` `Google Apps Script`
